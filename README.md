# AI Business Analytics Toolkit

AI-powered analytics platform for exploring business data, generating dashboards, and extracting insights using natural language.

## Overview

This project helps businesses understand their data faster by combining:

- data upload and preview
- automatic charts and KPIs
- AI-generated summaries
- natural language Q&A on datasets

## Problem

Many small businesses have data but lack the tools to quickly extract useful insights.

## Solution

This application turns raw CSV data into:
- visual dashboards
- business insights
- recommendations
- simple answers to natural language questions

## Features

- Upload CSV datasets
- Automatic data profiling
- KPI generation
- Chart recommendations
- AI-powered insights
- Ask questions about your data

## Tech Stack

### Frontend

- Tailwind CSS
- Recharts

### Backend
